# 最小構成リポジトリ分離計画: DEVA方式 SAM2+BEN2/TB ハイブリッド動画アプリ

- 調査日: 2026-07-01
- 対象: `gradio_app_sam2_ben2_tb_hybrid_deva_for_Movie.py` のみを親会社へ渡す前提の最小リポジトリ構成調査
- 前提: 本ファイルは調査記録であり、実際の新規リポジトリ作成・ファイルコピーはまだ実施していない（要別途承認）

## 1. 対象アプリの依存チェーン（コード確認済み）

```
gradio_app_sam2_ben2_tb_hybrid_deva_for_Movie.py
  └─ pipelines/route_a_deva_hybrid_video_pipeline.py（Haystack Pipeline 定義）
       ├─ pipelines/components/video_model_components.py
       │     ├─ VideoReader / VideoWriter / FrameSequenceWriter / TrackingOverlayWriter
       │     └─ SAM2VideoPropagator（SAM2.1 動画伝搬。checkpoints/SAM2/sam2.1_hiera_large.pt）
       ├─ pipelines/components/deva_semi_online_tracker.py（DevaSemiOnlineTracker）
       │     ├─ pipelines/components/detection_island.py（GroundingDINO 検出島）
       │     ├─ pipelines/components/consensus.py（IoU consensus マージ、純関数）
       │     └─ pipelines/components/video_model_components.SAM2VideoPropagator（再掲）
       ├─ pipelines/components/hybrid_alpha_components.py（BEN2TransparentHybridVideoExtractor）
       │     ├─ pipelines/components/ben2_components.py（BEN2Extractor）
       │     ├─ pipelines/components/model_components.py（TransparentBGExtractor / default_device）
       │     ├─ pipelines/components/route_a_common.py（config/route_a.toml 読込・alpha_to_rgba）
       │     ├─ pipelines/components/common.py（compose_alpha 等の純関数）
       │     └─ pipelines/components/video_common.py（frame/mask 共通処理）
       └─ pipelines/components/video_common.py（build_frame_mask_sequence）
```

- `pipelines/components/model_registry.py` と `config/inference_models.toml` はこのアプリでは**未使用**
  （detector/tracker は GroundingDINODetector・SAM2VideoPropagator の既定値を直接使用しており、
  モデル切替 UI は持たない）。
- MAM 本体（`networks/`, `dataloader/`, `evaluation/`, `trainer.py`, `mam_vit*.pth`, `segment-anything/`）は
  このアプリの実行経路に**含まれない**（DEVA/BEN2/TB ハイブリッド動画専用アプリのため）。

## 2. 最小ファイル構成（案）

親会社へ渡すリポジトリは「このアプリ 1 本が動く」ことだけを目的とし、以下のみを含める。

```
matting-anything-hybrid-video/
├── README.md                                  # セットアップ手順（本リポジトリ用に新規作成）
├── LICENSE                                     # 既存ライセンスを踏襲
├── .gitignore                                  # checkpoints/ outputs/ .venv/ 等を除外
├── .gitmodules                                 # samurai サブモジュール定義（sam2 fork取得用）
├── pyproject.toml                              # uv 用環境定義（torch cu124 index 含む）
├── uv.lock                                     # uv 環境ロックファイル（再現性担保）
├── requirements.txt                            # pip 運用時の代替手段（.venv + pip）
├── run_movie_apps.bat                          # 起動用バッチ（Windows）
├── gradio_app_sam2_ben2_tb_hybrid_deva_for_Movie.py   # ★対象アプリ本体
├── config/
│   └── route_a.toml                            # alpha/blur_guide/composite/deva パラメータ
├── pipelines/
│   ├── __init__.py
│   ├── route_a_deva_hybrid_video_pipeline.py
│   └── components/
│       ├── __init__.py
│       ├── video_model_components.py
│       ├── deva_semi_online_tracker.py
│       ├── detection_island.py
│       ├── consensus.py
│       ├── hybrid_alpha_components.py
│       ├── ben2_components.py
│       ├── model_components.py
│       ├── route_a_common.py
│       ├── common.py
│       └── video_common.py
├── GroundingDINO/                              # pip install -e 用ソース（重みは含まない, 約0.7MB）
├── samurai/                                    # git submodule（sam2 フォーク。sam2 パッケージ配布元）
│   └── sam2/                                   # `pip install -e samurai/sam2` 対象（約25MB）
├── checkpoints/                                # git管理外。README記載のDLリンクからユーザーが配置

        ├── test_route_a_deva_hybrid_video_pipeline_wiring.py
        └── test_routea_deva_hybrid_movie_sync_wiring.py
```

### 環境作成に必須のファイル（最低限）

| ファイル | 役割 |
|---|---|
| `pyproject.toml` + `uv.lock` | uv 運用時の環境再現（本アプリの想定実行環境） |
| `requirements.txt` | pip 運用時の代替（本リポジトリ既定の `.venv + pip` 方式） |
| `.gitmodules` + `samurai/`（submodule） | `sam2`（SAM2.1 動画伝搬）パッケージの取得元 |
| `GroundingDINO/`（サブディレクトリ、重みなし） | `groundingdino` パッケージのソース。CUDA拡張ビルドに必要 |
| `README.md`（新規作成） | conda/venv作成 → `pip install -r requirements.txt` →
  `pip install -e samurai/sam2` → `pip install -e GroundingDINO`（`BUILD_WITH_CUDA=True` 必須）→
  重み配置手順、を明記 |

### 除外して良いもの（対象アプリが使わないため）

- `networks/`, `dataloader/`, `evaluation/`, `trainer.py`, `main.py`, `Matting_Anything*.ipynb`,
  `Sam2_Transparent_Background_Haystack*.py/ipynb`, `Sam2_BEN2_RouteA_for_Movie.py/ipynb`,
  `gradio_app.py`, `gradio_app_haystack.py`, `gradio_app_sam2_ben2_route_a*.py`,
  `gradio_app_sam2_transparent_BG*.py`, `inference_benchmark.py`, `run_video_matting_headless.py`
- `segment-anything/`（MAM ViT-B/L/H 用。本アプリは SAM2 のみ使用）
- `checkpoints/mam_vit*.pth`（MAM 用重み。本アプリ未使用）
- `config/MAM-*.toml`, `config/inference_models.toml`（本アプリ未使用。ただし将来モデル切替UIを
  追加するなら `inference_models.toml` は有用。今回スコープ外）
- `assets/`, `archive/`, 日本語運用フォルダ（`エラーログ/セッションログ/報告書/計画書/調査/資料`）は
  社内運用ドキュメントのため非公開範囲（親会社へは README に必要事項のみ抜粋して渡す）
